# Advisor Plans

Handoff plans written by the architect specialist (`.claude/agents/architect.md`
via the `improve` skill) at the end of substantial work sessions.

Each plan is a self-contained, prioritized implementation brief meant to be
executed by another agent in a fresh session — it should name files, state
acceptance criteria, and not assume the author's conversation context.

Plans here are point-in-time snapshots. Once a plan's work lands, the
ROADMAP amendments log is the durable record; the plan file may be deleted.
